# Pré-processamento e limpeza de dados para modelagem preditiva

Este projeto contempla os processos necessários para preparação de dados para modelagem preditiva. Serão abordados os seguintes assuntos:

- Limpeza e transformação de dados
- Tratamento de valores ausentes
- Conversão de tipo de dados
- Identificação e tratamento de outliers

Este projeto considera os dados de uma empresa de telecomunicações, com cerca de 150 mil registros.
